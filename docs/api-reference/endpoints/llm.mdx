---
title: LLM Chat Completions
openapi: post /{provider}/chat/completions
---

OpenMind provides seamless integration with multiple Large Language Models (LLMs) through a unified chat completions API. This endpoint enables you to generate AI responses using various LLM providers while maintaining a consistent interface.

Key Features:
- Unified API across multiple LLM providers
- Standardized request/response format
- Provider-specific parameter support
