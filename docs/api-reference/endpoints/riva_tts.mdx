This endpoint converts text into natural-sounding speech using NVIDIA's Riva Text to Speech (TTS) model. The service provides high-quality speech synthesis with support for multiple voices and languages.

### Overview

The Riva TTS endpoint allows you to:
- Generate human-like speech from plain text
- Choose from multiple voice options
- Control speech parameters like speed and pitch
- Receive audio output in various formats

For optimal results, ensure your input text is well-formatted and contains proper punctuation.
