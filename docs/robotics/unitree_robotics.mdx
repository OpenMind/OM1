OM1 can control a Unitree Go2 Air. This has been tested on macOS Sonoma 15.2.

### Step 1 - Game Controller Setup

Connect an Xbox controller to your computer.

<Tabs>
  <TabItem value="mac" label="macOS">
    Install `hidapi`:
    ```bash
    brew install hidapi
    ```
    
    **Note:** Due to a known bug with `brew` and `uv`, if you encounter errors like `Unable to load any of the following libraries:libhidapi-hidraw.so`, add this to your `.zshrc`:
    ```bash
    export DYLD_FALLBACK_LIBRARY_PATH="$HOMEBREW_PREFIX/lib"
    ```
  </TabItem>
  
  <TabItem value="linux" label="Linux">
    ```bash
    sudo apt-get update
    sudo apt-get install python-dev libusb-1.0-0-dev libudev-dev libhidapi-dev
    
    # If needed:
    sudo pip install --upgrade setuptools
    sudo pip install hidapi
    ```
  </TabItem>
</Tabs>

### Step 2 - Network Configuration

1. Connect the Unitree Go2 Air to your computer via Ethernet
2. Configure the network adapter:
   - Open network settings
   - Locate the interface connected to Go2 Air
   - Set IPv4 mode to `Manual`
   - IP Address: `192.168.123.99`
   - Subnet Mask: `255.255.255.0`
   - Apply changes and wait for reconnection
3. Add to your `.env` file:
   ```
   UNITREE_WIRED_ETHERNET=eno0  # Replace with your adapter name
   ```
   For simulation mode, use: `UNITREE_WIRED_ETHERNET=SIM`

### Step 3 - CycloneDDS Setup

Install CycloneDDS:
```bash
git clone https://github.com/eclipse-cyclonedds/cyclonedds -b releases/0.10.x
cd cyclonedds && mkdir build install && cd build
cmake .. -DCMAKE_INSTALL_PREFIX=../install -DBUILD_EXAMPLES=ON
cmake --build . --target install
```

Add to your `.zshrc` or equivalent:
```bash
export CYCLONEDDS_HOME="$HOME/cyclonedds/install"
```

### Step 4 - Python Dependencies

Install the DDS module:
```bash
uv pip install -r pyproject.toml --extra dds
```

### Step 5 - Robot Control

Launch the robot controller:
```bash
uv run src/run.py robot_wallet_safe
```

#### Controller Commands
| Button | Action |
|--------|--------|
| A | Stand up |
| B | Sit down |
| X | Shake paw |
| Y | Stretch |

<Warning>
Advanced movements like `move`, `pounce`, and `run` require additional implementation. Exercise caution when testing these features, especially around animals or children.
</Warning>

### Troubleshooting

#### Channel Factory Init Error
Verify your network interface adapter name in `.env`. This should be the adapter on your development machine that connects to the Unitree.

#### CycloneDDS Library Not Found
- Ensure CycloneDDS is installed
- Verify `CYCLONEDDS_HOME` is set correctly

#### Connectivity Issues
Test basic connectivity:
```bash
# Test network connection
ping 192.168.123.99

# Test CycloneDDS (from cycloneDDS/build)
# Terminal 1:
./bin/RoundtripPing 0 0 0

# Terminal 2:
./bin/RoundtripPong
```

<Tip>
macOS users will need to allow incoming connections for RoundtripPing and RoundtripPong in the security settings.
</Tip>

If these tests pass but issues persist, please open a GitHub issue for support.
