## Overview

[x402](https://www.x402.org/) is an open protocol for internet-native payments. It is a simple, secure, and interoperable protocol that allows users to send and receive payments from anywhere in the world.

## Integration

OM1 has a built-in connector for x402. You can use it by adding the following to your config:	

Here is an example of x402 as input:

```
  "agent_inputs": [
    {
      "type": "X402Input"
    }, ... ...
  ]
```

Here is an example of x402 as action:	

```
  "agent_actions": [
    {
      "name": "x402_command",
      "llm_label": "move",	
      "implementation": "passthrough",
      "connector": "x402"
    }, ... ...
  ]
```
